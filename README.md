# Image Classification using CNN

This project is a deep learning image classification system created with Convolutional Neural Networks (CNN). The model is trained to classify images, such as dogs and cats, and offers predictions through a simple web interface.

---

## Features

- Image classification using CNN
- Deep learning model built with TensorFlow/Keras
- Web interface for uploading images
- Real-time prediction
- Clear project structure for growth

---

## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- Flask
- HTML
- CSS
- NumPy

## How to Run the Project

1. Clone the repository

git clone https://github.com/yourusername/ImageClassification.git

2. Install required libraries

pip install tensorflow flask opencv-python numpy

3. Train the model

python train_model.py

4. Run the application

python app.py

5. Open your browser

http://127.0.0.1:5000/
